# Cluster D + Transformation Cluster Governance

## The six archetypes

| Arcana | Core mechanism |
|--------|---------------|
| 12 Hanged Man | developmental suspension — pause before reorientation |
| 13 Death | irreversible ending — identity closure |
| 14 Temperance | emotional regulation — intensity metabolization |
| 15 Devil | unconscious compulsion — function-driven repetition |
| 16 Tower | structural collapse — false-certainty disclosure |
| 18 Moon | perceptual distortion — fear-filled gap-filling |

## Critical boundary pairs

### Suspension vs Ending (12 vs 13)
- Hanged Man: situation has not ended, has not collapsed — forward motion pauses while framework shifts
- Death: something has genuinely concluded — must be closed, not extended or suspended

### Collapse vs Reorientation (16 vs 12)
- Tower: external or structural crisis discloses false foundations
- Hanged Man: no collapse — developmental pause by internal requirement

### Compulsion vs Regulation (15 vs 14)
- Devil: pattern continues because it serves an unconscious function
- Temperance: intensity cycling between explosion and suppression

### Ambiguity vs Inversion (18 vs 12)
- Moon: fear fills perceptual gaps with threat content
- Hanged Man: forward motion suspended so that orientation can shift — not a perception problem

### Withdrawal vs Developmental Pause (9 vs 12)
- Hermit: chosen, renewable, permanent orientation mode
- Hanged Man: temporally bounded, developmentally specific, not introverted by nature

## Governance rule

Any content about stuckness, uncertainty, waiting, ambiguity, freeze, or inactivity
must be assigned to the correct archetype before being written.
Do not default to Hanged Man as the catch-all for non-movement states.

## Suspension vs Non-Movement Differentiation Matrix

| Archetype | Why movement stops | Mechanism | Resolution |
|-----------|-------------------|-----------|------------|
| 12 Hanged Man | current framework cannot continue intact | developmental suspension — perspective inversion required | orientation shift, not more information |
| 14 Temperance | intensity exceeds processing capacity | explosion/suppression cycling | regulation — expanding window of tolerance |
| 15 Devil | pattern serves an unconscious function | compulsive repetition — function not yet identified | function identification — building alternative delivery |
| 18 Moon | ambiguity filled with fear content | perceptual distortion — gap-filling from anxiety | calibration — distinguishing reading from generating |
| 9 Hermit | deliberate inward orientation | chosen renewable solitude — not transitional | not a problem to solve — a permanent mode |

## Anti-confusion rule

If a page about non-movement, withdrawal, or delayed action is being written:
1. Identify which mechanism is operating from the table above
2. Write only to that mechanism
3. Do not blend mechanisms from multiple rows
4. Reference this table in the governance notes of any new arcana page touching these themes

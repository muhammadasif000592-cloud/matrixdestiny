## How to use this file

Before writing any arcana page:
1. Check if the arcana you are writing has a reference entity here
2. If yes — match that standard exactly
3. If no — after approval, add the new arcana here as a reference entity
4. Never let forbidden bleed terms appear in a protected entity's page

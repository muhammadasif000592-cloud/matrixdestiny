# Generation 2 Language Patterns — matrixdestiny.co

**Purpose:** Prevent lexical signature drift across arcana pages. Each page must read as its own document — not as a variation on a shared template. Consulted during writing and pre-publication review.

**Scope:** All Generation 2 arcana content in all three languages (EN, PT-BR, UK).

---

## Section 1 — Overused Phrases: Limit to Once Per File

These phrases have become structural defaults across Generation 2 pages. If a phrase appears, it has used its allocation for that file. Do not repeat it in the same file, and seek alternatives before using it in new files.

| Phrase | Alternative directions |
|---|---|
| "the mechanism sustaining it" | "what keeps it running," "the underlying driver," "what maintains the cycle," "why it persists" |
| "the template stays active" | "the pattern carries forward," "the original response keeps running," "it persists into," "the adaptation continues operating" |
| "the constant across" | "the element present in every case," "what doesn't change," "the variable that remains" |
| "below the level of conscious navigation" | "outside deliberate awareness," "operating before conscious choice," "encoded below what reflection normally reaches," "not available to direct examination" |
| "the original conditions no longer exist" | "those circumstances have passed," "the environment it was built for is gone," "this context is different from the one that produced it" |
| "precise adaptation to the environment" | "a functional response to those conditions," "what made sense in that context," "an intelligent adjustment to the original situation" |
| "carried into adult contexts" | "persisting into adulthood," "brought forward into current relationships," "continuing to operate in situations that no longer match the original" |

**Application rule:** If two or more of these phrases appear in a single file, flag for revision before publishing. The goal is not to ban the ideas — it is to force more precise expression each time.

## Section 2 — Per-Arcana Lexical Territories

Each arcana owns a specific vocabulary. Using another arcana's primary vocabulary within a page is a domain leak — it signals that content has drifted into the wrong psychological territory.

### Arcana 8 — Strength

**Owns:** power, authority, delegation, dominance, submission, control, calibration, self-assertion, proportion, hierarchy, surrender, deference, yielding, position, territory, jurisdiction, weight

**Primary metaphors:** binary, poles, the third mode, the grip that tightens, the position held or released

**Exclusion list — do not use in 8's pages:**
- feedback, adjustment, contribution (→ Justice, arcana 11)
- regulation, intensity, explosion, suppression, nervous system, tolerance, window (→ Temperance, arcana 14)
- guilt, accountability, pattern repetition, karma debt (→ Justice, arcana 11)

**Diagnostic check:** If a sentence about arcana 8 could be lifted and placed in a Justice page without revision, it has leaked.

---

### Arcana 11 — Justice

**Owns:** consequence, reciprocity, proportion, contribution, accountability, repair, feedback, adjustment, pattern, cycle, cause, effect, recognition, correction, reading, accuracy, investigation

**Primary metaphors:** the loop that persists, the constant that isn't seen, the incomplete inquiry, the distributed weight

**Exclusion list — do not use in 11's pages:**
- nervous system, window of tolerance, explosion, suppression, somatization (→ Temperance, arcana 14)
- dominance, submission, authority, delegation, control as a structural concept (→ Strength, arcana 8)
- shadow, compulsion, unconscious pattern (→ Devil, arcana 15)

**Diagnostic check:** If a sentence explains emotional intensity rather than pattern accountability, it belongs to arcana 14, not 11.

---

### Arcana 14 — Temperance

**Owns:** regulation, intensity, suppression, explosion, nervous system, tolerance, window, integration, somatization, dissociation, activation, fuel, channel, metabolize, impulse, pressure

**Primary metaphors:** the window of tolerance, fuel vs. obstacle, the pressure that builds and releases, channeling vs. battling

**Exclusion list — do not use in 14's pages:**
- power, authority, dominance, delegation, calibrated position (→ Strength, arcana 8)
- accountability, consequence, contribution, pattern repetition from unowned choices (→ Justice, arcana 11)
- compulsion, shadow, unconscious function (→ Devil, arcana 15)

**Diagnostic check:** If a sentence attributes an emotional reaction to a power dynamic rather than to nervous-system intensity, the subject may belong to arcana 8 or 11.

## Section 3 — Prohibited Abstract Sentence Structures

These patterns produce semantic stiffness. They are the Generation 2 equivalents of the Generation 1 problem of describing symbols instead of mechanisms. Flag and rewrite on any page where they appear.

### 1. Overuse of "X is not Y, it is Z"

**Problem:** One use of this structure per file is powerful. Three or more make the page feel like a series of corrections rather than a description.

**Bad example:**
> Justice is not morality, it is accuracy.
> Justice is not punishment, it is feedback.
> Justice is not about being good, it is about seeing clearly.

**Fix:** State the affirmative directly. Pick the one inversion that earns its place.

---

### 2. Generic childhood adaptation paragraphs

**Problem:** Karmic origin sections that could appear on any arcana page. The signal: if you removed the arcana name and the specific behavioral domain, the paragraph would be indistinguishable from another arcana's karmic section.

**Red flags:**
- "early experiences shaped the person's relationship with..."
- "in environments where [vague difficulty], the person learned to..."
- "this protective strategy made sense then but no longer serves now"

**Fix:** Name the specific structural feature of the environment. "Binary power distribution" (arcana 8). "Arbitrary or inconsistent cause-and-effect" (arcana 11). "Environments where emotional intensity was punished or dangerous" (arcana 14). These are different environments with different textures — the writing must reflect that.

---

### 3. "The work is not to..." closings

**Problem:** Almost every karmic section ends with this construction. It has become the default resolution frame and is losing its impact.

**Audit:** Search each file for "the work is not." If it appears, replace with a more concrete direction.

**Bad closing:**
> The work is not to choose the opposite pole but to develop the third mode.

**Better direction:**
> Building the third mode means practicing the specific action — not the general disposition. Bounded delegation. Low-stakes assertions. The practices that build tolerance for the mode that currently produces anxiety.

---

### 4. Resolution framing in closing paragraphs

**Problem:** Final paragraphs that end with clean direction statements — "the work is X," "the goal is Y," "this is what changes" — repeat across pages and produce a uniform cadence that signals template, not voice.

**Fix:** Vary closing structure by arcana. Not every page needs to end with a direction. Some pages can end with the observation itself. Let the pull quote do some of the closing work.

---

### 5. Syntactic uniformity warning

The following phrase structures are becoming Generation 2 defaults. Use maximum once per file, not as opening anchors:
- "the mechanism by which..."
- "the process through which..."
- "the specific point where..."
- "the capacity to..."
- "the template stays active"
- "the original conditions no longer exist"

Each arcana must find its own opening rhythm.

## Section 4 — Stylistic Diversification Rules

Each arcana must have a distinct opening rhythm, pull quote style, and FAQ register. If two pages feel like they were written by the same writer on the same day, audit for structural similarity.

### Opening rhythm (Section 5: "What [Name] describes")

**Must vary across pages.** Three acceptable patterns — use each sparingly:

1. **Structural observation about the human condition** — opens with a universal fact about how people operate, then introduces the arcana as governing one dimension of that fact. (Used in arcana 8 EN: "Every person exists within structures of authority...")
2. **Mechanism-first** — opens with the psychological mechanism directly, no preamble. ("Justice doesn't govern morality. It governs accuracy.")
3. **Contrast** — opens by naming what the arcana is not, then defining what it is. Use sparingly — see Section 3.1.

No arcana page should open with "Arcana N in the Matrix of Destiny represents..." as the first sentence of the body. This belongs to the Definition Block (frontmatter description), not the body.

---

### Pull quote style

Pull quotes should not all follow the same "X is not Y — it is Z" or "Both poles are..." pattern. Diversify:

- **Mechanism statement** — one sentence that names how the block operates
- **Behavioral contrast** — names what the active state can do that the blocked state cannot
- **Origin reframe** — shifts how the reader sees the karmic source (used at end of karmic section)
- **Observer reframe** — what someone watching from the outside would see but the person inside can't

Limit: no arcana page should have both pull quotes follow the same pattern.

---

### FAQ register

FAQ answers must not all follow the same sentence rhythm. Vary by question type:

- **Orientation question (Q1):** Can open with "indicates that" but the rest of the answer must include specific behavioral description, not just state/pole labeling
- **Mechanism question (Q2):** Should read like an explanation, not a list — use connective logic rather than parallel structure
- **Precision question (Q3):** Must name a behavioral test the reader can apply — not just a conceptual distinction
- **Scope question (Q4):** Should lead with the most surprising or counterintuitive domain manifestation
- **Action question (Q5):** Must be pole-specific; the dominance/exemption-mode person needs different practices than the submission/over-attribution-mode person — generic action advice is a fail
